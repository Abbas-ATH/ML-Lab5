# Lab 5 – Feature Engineering (Classification)

In this lab, we applied feature engineering techniques to improve a classification model that predicts Order_Status.

First, we created new engineered features such as price differences and time-based features (day, month, year). These features help the model capture patterns related to price movement and time.

Then, we trained a Random Forest classifier and evaluated its performance using accuracy, classification report, and confusion matrix.

The model achieved very high accuracy, and we analyzed the most important features influencing the predictions. We also applied feature selection using SelectFromModel to simplify the model while maintaining similar performance.
